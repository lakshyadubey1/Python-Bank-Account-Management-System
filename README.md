# Python Bank Account Management System

## Overview

This is a simple Python-based Bank Account Management System. The project allows users to create bank accounts, perform transactions, view account details, generate reports, and save data using file handling.

## Features

* Create new bank accounts
* View account details
* Deposit money
* Withdraw money
* Transfer funds between accounts
* View transaction history
* Generate basic reports using NumPy
* Store account and transaction data using Pickle

## Technologies Used

* Python
* Pickle
* NumPy

## How to Run

1. Clone the repository.
2. Open the project in your preferred Python environment.
3. Install NumPy if not already installed:

   ```
   pip install numpy
   ```
4. Run the Python file:

   ```
   python bank_management.py
   ```
